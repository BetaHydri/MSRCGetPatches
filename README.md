# Get-ActualCVEsByProduct 
Enumerates all CVEs for a specified product for a specified year-month combination

PARAMETER ***ProductTitle***<br />
* `"Windows Server 2016*"` <br />
* `"Windows Server 2016"` <br />
* `"Microsoft Edge*"`<br />

PARAMETER ***OutputStyle***<br />
* `HTML`
* `GridView`

PARAMETER ***Date***<br />
* `2021-Jun`
* `2023-Jan`
